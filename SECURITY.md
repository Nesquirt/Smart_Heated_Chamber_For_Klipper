# SECURITY

## Security Policy

Smart Chamber Heater has been designed with safety as a primary
objective. However, **software alone cannot guarantee the safe operation
of a heating system**.

The user is solely responsible for verifying that the hardware,
electrical installation and printer configuration are suitable for the
intended application.

------------------------------------------------------------------------

# Supported Versions

Security fixes will be applied only to the latest stable release unless
otherwise stated.

------------------------------------------------------------------------

# Reporting Security Issues

If you discover a safety-related bug or vulnerability that could lead to
unsafe printer operation, please report it privately before opening a
public issue.

Please include:

- Printer model
- Klipper version
- Hardware configuration
- Chamber heater type
- Thermal fuse rating
- Relevant configuration
- Steps to reproduce
- Logs (if available)

------------------------------------------------------------------------

# Hardware Requirements

Before using Smart Chamber Heater, verify that:

- A correctly rated thermal fuse is installed.
- Wiring complies with local electrical regulations.
- All mains-voltage wiring is properly insulated.
- The heater is suitable for continuous operation.
- Temperature sensors are correctly installed and calibrated.
- Fans operate in the correct direction.
- Emergency shutdown has been tested.

Never operate a chamber heater unattended until all safety systems have
been verified.

# Best Practices

Always:

- Test from room temperature.
- Verify every protection individually.
- Monitor the first heating cycles.
- Inspect wiring regularly.
- Replace damaged components immediately.
- Keep a fire extinguisher suitable for electrical fires available in
  the workshop.
